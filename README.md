# Data Engineering Journey 🚀

Documenting my path into data engineering — notes, exercises, and projects as I work through the course, in public.

## About Me

I'm a data analyst with a background in capital/money markets monitoring and pharmaceutical IT, working with SQL, Python (Pandas, NumPy, Matplotlib), Excel/VBA, Power BI, and R. This repo is where I'm extending that foundation into core data engineering skills: pipelines, orchestration, distributed processing, and infrastructure.

## Why This Repo

- Build a public, dated record of what I'm learning
- Force myself to write things down clearly (best way to know if I actually understood it)
- Have a portfolio of small projects that show the work, not just claim it
